## Juego de Adivinanza Python
Este proyecto es un juego sencillo desarrollado en Python donde el usuario debe adivinar un número secreto generado aleatoriamente entre 1 y 99. El jugador cuenta con un máximo de 10 intentos para acertar. En cada intento, el programa indica si el número ingresado es mayor o menor que el número secreto, ayudando a orientar la siguiente elección. El juego finaliza cuando el usuario adivina el número o se agotan los intentos.

Este proyecto es ideal para practicar conceptos básicos de Python como bucles, condicionales, entrada de datos por consola y uso de la librería random.
